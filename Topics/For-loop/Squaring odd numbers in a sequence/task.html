Squaring odd numbers in a sequence

Hey there! This problem might be a bit unpredictable, but give it a go and let us know how you do!
You are given a single integer n. Please write a program in Java language that iterates n times and prints the square of the current iteration number if it's an odd number; otherwise, print the current iteration number itself. The square should be calculated using a ternary operator. You should use the for-loop and increment constructs. The output should be a list of n numbers with each number on a new line.
Sample Input:
5

Sample Output:
1
2
9
4
25


Sample Input:
3

Sample Output:
1
2
9



Memory limit: 256 MB
Time limit: 5 seconds
